Netflix Dashboard Project

This project presents a detailed visual analysis of Netflix’s content catalog using a dashboard built from a dataset of movies and TV shows. 
It explores trends across directors, ratings, content types, and genres, helping understand how Netflix’s library has evolved over time.
The dataset includes 12 columns such as title, director, cast, country, rating, and genre. The dashboard allows for interactive filtering and analysis.

Dataset Overview

This project uses the Netflix Titles dataset with the following columns:

Column	Description
show_id	Unique ID for each title
type	Content type (Movie or TV Show)
title	Name of the content
director	Director(s)
cast	Main actors
country	Country of origin
date_added	Date the content was added to Netflix
release_year	Year of original release
rating	Content rating (TV-MA, PG, etc.)
duration	Length in minutes or number of seasons
listed_in	Genre/category
description	Brief summary of the title

Dashboard Features & Insights
Director Count by Release Year
Track how many directors contributed to Netflix content over time.

Content Added by Rating
See how different content ratings (e.g., TV-MA, PG) were added over time.

Rating Count by Day
Analyze how content ratings are distributed across the days Netflix adds content.

Content Type by Country
Compare the number of Movies vs TV Shows produced by each country.

Rating by Genre (listed_in)
Understand how different genres are distributed across content ratings.

Key Takeaways

Netflix has seen a consistent rise in new directors contributing each year.

TV-MA and TV-14 are the most commonly added ratings.

Drama and International genres dominate across all ratings.

The U.S., India, and the U.K. are leading content contributors.
